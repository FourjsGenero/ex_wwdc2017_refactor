Example for Genero 3.10 EAP using various features to renovate a
simple HR application. The example runs from a sqlite database in
$(ProjectDir)/data/hr.db

Features used in the example include:

Import FGL modules
* Multiple Dialogs
* SubForms
* SubDialogs
* On Change
* On Fill Buffer
* On Sort
* TopMenus
* Toolbars
* Function References
* Aggregate fields
* Auto Generate Documentation
* Placeholders
* Try / Catch
* Dictionaries

UX enhancements include:
* Universal Search module
* Contextual Save
* Infinite arrays
* Mandatory field indicators
* Ready to Edit
